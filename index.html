News publishing site

Description

This site allows multiple authors to publish articles on a public portal

Used Technologies

  • Spark Framework
  • PostgreSQL
  • Velocity template engine
  • Bootstrap

Features:

  • User System - Allows for the creation of different users
  • Permits system - During user creation their permits are drfined
  • Data persistence - Articles, users and password are stored in a PostgreSQL Database
  • Password Encription - Sha1 Password Encryption
  • Articles Creation - Users can publish articles that are displayed in the main site which is accessible by anyone

Type-Bit

Description

This game was developed as a team project at the end of the first class of the school I attended. It is a Typing Game, the player has to type the words which appear on screen as fast as possible. There are 8 levels of increasing difficulty. The code is simple but it was very fun to develop and we were really happy with the final result. You can play it by pressing the button at the bottom of the box, try it, it's fun!

Used Technologies

  • JavaScript
  • JQuery
  • Bootstrap

Features:

  • Music is used to give the player a feeling of urgency. The closer the player gets to running out of time, the faster the music is played.
  • To give the user a visual feedback about how much time he has left, the background changes from dawn till dusk.
  • 8 levels. The higher the level, the longer the words to type.
  • Score system which gives and takes point when words are correctly or wrongly typed.
  • Very hard to beat, players need to be really fast to beat the last levels.
