News publishing site

Description

This site allows multiple authors to publish articles on a public portal

Used Technologies

  • Spark Framework
  • PostgreSQL
  • Velocity template engine

Features:

  • User System - Allows for the creation of different users
  • Permits system - During user creation their permits are drfined
  • Data persistence - Articles, users and password are stored in a PostgreSQL Database
  • Password Encription - Sha1 Password Encryption
  • Articles Creation - Users can publish articles that are displayed in the main site which is accessible by anyone
